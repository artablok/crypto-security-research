## 🛡️ How to Protect Yourself

1. **DO NOT** interact with unknown tokens
2. **DO NOT** click links in token descriptions
3. **DO NOT** attempt to sell or transfer scam tokens
4. **HIDE** the tokens in your wallet (Trust Wallet: tap token → Hide)
5. **REPORT** to wallet support and community
